# PianoNumberScales
Simple 'flashcard' game in Python for learning the musical number scales/system.


## Usage

`python run.py <flash_interval> <scale>`

eg: 

    python run.py 2 C
    
    python run.py 3 c#
    
    python run.py 1 Bb

## Sample output
python run.py 3 f#

   5th tone of F#:   C#
   
   1st tone of F#:   F#
   
   6th tone of F#:   D#
   
   4th tone of F#:   B
   
   1st tone of F#:   F#
   
   7th tone of F#:   E#
